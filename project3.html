Heya is currently working on this project ... this page will be updated soon!

OBJECTIVE: Build a full-stack Ruby in Rails application that uses external APIs to help users make decisions on what to eat, what to watch, etc.


WHAT I LEARNED: How to make React *IN* Rails website using Devise, How to connect an external API, & How to work in a remote team of four using Zoom, Trello, and Github


WHAT I USED: React, Rails, HTML/CSS, Figma, Trello, Github, external API (imdb)


USER STORY:

- As a user, I can go to the landing page and select a category of decisions I need to make.

- As an unregistered user, I can sign up.

- As a registered user, I can log in.

- As a logged in user, I can click on a decision category and get a random decision generated.

- As a logged in user, I can see my decision history with my notes and whether they were saved as favorites.
