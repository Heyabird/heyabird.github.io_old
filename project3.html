SET UP NOTES

SUMMARY: PikaMovie is a React on Rails app that chooses a movie for users according to their genre preferences; users can log in to store selected movies and comment on them.

OBJECTIVE: Make an app that tackles the modern day "Decision Fatigue" by helping users make decisions on what to watch.

WHAT I LEARNED:

- How to make a React *IN* Rails application using Devise authentication/authorization

- How to connect to an external API

- How to work in a remote team of four with Agile framework using Zoom, Trello, and Github

WHAT I USED: React, Rails, HTML/SCSS, Figma, and an external API (The Movie Database)

HOW I SET IT UP: Click HERE to see my notes on how I set up this project!

COLLABORATORS: Meo Hyunh, Jonas Fernandes, Liz Havird

USER STORY:

**MVPS are boldened

1. As an unregistered user, I see a landing page with the name of the app as a header, a nav bar, and a button "what to watch".

2a. As an unregistered user, I can see a sign-up and log-in button in the landing page. I can click on a sign-up button to sign-in.

2b. As a registered user, I click on a log-in button to log in.

3. As an unregistered user, I can see the button: “What to watch” in the landing page.

4. As an unregistered user, I can click on “what to watch”

5. As an unregistered user, when I click on "what to watch", I am asked "what genres you want?", and I can choose a genre or genres I want to filter the movies from.

6. As an unregistered user, I can click on a "submit" to receive a randomly generated movie.

7. As a registered user, I can see and click on a logout button to log out.

8. As a registered user, I can click on 'decision history' in the navbar to see an index of movies that were previously selected for me.

9. As a registered user, I can click on individual movies to go to a history show page. In there, I can view the details about the movie and have the option to favorite decisions suggested to me in the past. I also see a button to take me back to the history index page.

10. As a registered user, I can click on 'favorites' in the nav to see an index of movies I saved as my favorite. There is a button under each movie that I can click to delete it from the favorites index page.

11. As a registered user, I can click on one of the movies in the favorites index page to take me to favorite show page where I can see more details of each individual movie. Once on that page, I can add and edit a comment on the movie. I can also delete it from favorite or go back to favorites index page.

12. As a registered user, I can always go back to the home page.

13. As a user, I can click on a button to see the About Us page that tells the user how cool we are and links to our github profiles.
